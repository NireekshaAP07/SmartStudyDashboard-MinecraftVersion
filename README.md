 ⛏️ Smart Study Dashboard – Minecraft Edition

A **gamified productivity dashboard** that turns studying into a Minecraft-style experience.
Complete tasks, run focus sessions, earn XP, level up, and build your own block world 🧱✨

---

 🚀 Features

* 📝 **Task Manager** – Add, complete, and track study tasks
* ⏱️ **Custom Focus Timer** – Set your own study duration
* ⭐ **XP & Level System** – Earn XP for completing tasks and sessions
* 📊 **Stats Dashboard** – Track productivity and progress
* 📅 **Daily Completion Bonus** – Extra XP for finishing all tasks
* 🧱 **Block Builder** – Build your Minecraft-style house as you progress
* 🔔 **Toast Notifications** – Instant feedback for actions
* 💾 **Local Storage** – Saves progress automatically

---

 🎮 How It Works

1. Add tasks to your planner
2. Complete tasks → earn XP ⭐
3. Run focus sessions → gain bonus XP ⏱️
4. Level up and unlock progress 🚀
5. Build your block house as a visual reward 🧱

---

 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Custom + Tailwind)**
* **Vanilla JavaScript**
* **LocalStorage (for persistence)**

 ⚙️ Setup & Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/NireekshaAP07/SmartStudyDashboard-MinecraftVersion.git
   ```

2. Open the project:

   ```bash
   cd SmartStudyDashboard-MinecraftVersion
   ```

3. Run:

   * Open `index.html` in your browser

No build tools or dependencies required ✅
